# IFJ compiler 2025
 This is a source code of compiler for semestral project from IFJ 2025 lightly based from language Wren.

 Building from source
 -------

 **GNU/Linux** based OS:
 1. Install dependencies: GNU Make, gcc (preferrably 12.5.0 as on BUT FIT Merlin server), Doxygen (optional)
 2. Inside `/src` directory run `make`. Built binary will be inside `build` directory in project root.
 3. To create Doxygen documentation, inside `/src` dir run `doxygen Doxyfile`. Documentation can be then viewed by <br>
    opening `html/index.html` inside `doc` directory from project root.
